# 🌐 Developer Portfolio

My personal developer portfolio built with **React** and deployed on **Vercel**.

🔗 **Live Site:** [portfolio-devrubey.vercel.app](https://portfolio-devrubey.vercel.app)

---

## 🛠️ Tech Stack

- **React** — UI framework
- **JavaScript** — logic
- **CSS** — styling with custom animations
- **Vercel** — deployment

---

## ✨ Features

- Smooth scroll navigation
- Animated sections on scroll
- Dark techy aesthetic
- Project cards linked to GitHub repos
- Contact section with email and GitHub links
- Fully responsive on all screen sizes

---

## 🚀 Running Locally

```bash
git clone https://github.com/devRubey/portfolio.git
cd portfolio
npm install
npm start
```

Opens at `http://localhost:3000`

---

## 📦 Sections

- **Home** — introduction with animated terminal typewriter
- **About** — background and experience
- **Projects** — all projects with GitHub links
- **Skills** — full tech stack
- **Contact** — email and GitHub
